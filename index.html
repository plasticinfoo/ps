PS

Polipropileno

El poliestireno es un plástico versátil usado para fabricar una amplia variedad de productos de consumo. Dado que es un plástico duro y sólido, se usa frecuentemente en productos que requieren transparencia, tales como envases de alimentos y equipos de laboratorio.

Se le conoce comúnmente como Styrofoam. Se usa para hacer vasos, platos, tazas, recipientes desechables, bandejas de carne y muchas cosas de un solo uso. El poliestireno es conocido por filtrar el estireno que puede causar daños al sistema nervios y está relacionado al cáncer. Utilizar estos envases para comidas calientes como el café podría ser de gran detrimento para la salud.

Este es el típico plástico que utilizan para elaborar las cajas de hamburguesas de los restaurantes de comida rápida. Es altamente contaminante por lo que no debe ser reutilizado para usos alimenticios, si evitas su uso estarás ayudando de gran manera al medio ambiente.
